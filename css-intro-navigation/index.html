Create navigational links within a website

There are several techniques to navigate within a website. The semantic tag for navigation is the <nav> tag. You can navigate throught your web site by using relative addressing. For our examples, we will use the unordered listing tag (<ul>).

  • page to page within the same folder
  • page to page between folders
  • staying on the current page
  • going to a specific place on the current page (on page navigation)

Navigational menus are placed in your <header> semantic area. They may also be place in your <footer> area as a sitemap. The nav tag like other semantic tags should have a <hx> tag. HOWEVER this header is optional. If you omit the header tag you will receive a document outliner errpr: Untitled Section. Until you are shown how to avoid error message, it will be acceptable on your document outline.
